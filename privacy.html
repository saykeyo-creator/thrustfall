Privacy Policy

Last updated: March 7, 2026

Thrustfall is a game developed by Lakes Games. This policy explains what data the game collects and how it is used.

Data We Collect

Thrustfall does not collect, store, or transmit any personal data to external servers. Specifically:

  • No accounts or sign-ups — the game does not require registration.
  • No analytics or tracking — no third-party analytics, advertising SDKs, or tracking pixels are used.
  • No cookies — the game does not set browser cookies.

Local Storage

The game stores gameplay preferences and progress (player name, settings, XP, cosmetic purchases) locally on your device using your browser's localStorage. This data never leaves your device and is not accessible to Lakes Games or any third party.

Multiplayer

When you play multiplayer, the game connects to a WebSocket server. The only data transmitted is:

  • Your chosen player name (not verified against any identity)
  • Game inputs (thrust, rotation, fire commands)
  • Cosmetic selections (ship skin, trail, engine sound, kill effect)
  • Equipped perks

This data is used solely to run the game session and is not stored after the session ends. No IP addresses are logged or retained.

Children's Privacy

Thrustfall does not knowingly collect any personal information from children. The game does not require or request personal information from any user.

Third-Party Services

The game is hosted on Render. Render's infrastructure may process connection metadata (IP addresses) as part of standard web hosting, subject to Render's own privacy policy.

Changes to This Policy

If this policy changes, the updated version will be posted at this URL with a new "Last updated" date.

Contact

If you have questions about this privacy policy, contact Lakes Games via the GitHub repository: github.com/saykeyo-creator/thrustfall
